Readme for: Citral CodeChef Level 1
Date: Sept 1, 2020

App requirements:

- To run this program, you are required to have Python 3.7 or later installed in your machine.
- If you do not, you can get it from: https://www.python.org/downloads/windows/
- User must also install below Python libraries: 
	[1] numpy
	[2] pandas
	[3] scipy
	[4] dash
	[5] dash_html_components
	[6] dash_core_components
	[7] dash_bootstrap_components
	[8] plotly-orca

Run instructions: 

1. Simply double-click "CitralCodeChefL1.bat". This program will run on your preferred internet browser.

2. 'ALGORITHM INFO' : View brief info on Wax Loop algorithm Level 1

3. 'SELECT TAB, WAX & DATASET FILES' : Select all 3 input files:
	- Dataset Level 1.xlsx
	- Dead Oil - Dulang 44 to 35C - OLGA tab - fixed format.tab
	- Dead Oil - DULANG 44 to 35C - OLGA WAX.wax
	This step is compulsory.

4. 'UPLOADED FILE LIST' : Double check the list of uploaded files.

5. 'OPEN USER DEFINED INPUTS' : Default parameters including diffusion rate calculation method are 
already pre-defined. Nonetheless, user can modify any of the pre-defined parameters and select diffusion 
rate calculation method from the 'dropdown menu'.

6. Click 'RUN' to start calculation and show results.
